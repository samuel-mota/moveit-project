## Tecnologias utilizadas
- Next.JS
- React.JS
- Typescript

## Como rodar localmente
1. Faça o clone do projeto 
2. Instale as dependencias
```bash
yarn
```
3. Rode o server
```bash
yarn dev
```
4. Abra o navegador no endereço:
http://localhost:3000
